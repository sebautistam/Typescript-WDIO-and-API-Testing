# Installations

Run `npm install` from root.

This project uses EditorCondfig to standarize text editor configuration.

# Test Execution Commands

Execute the complete test suite using `npm run wdio`, or `npm run test`.

Execute the tests for the User Registration feature using `npm run test:registration`.

Execute the tests for the User Login feature using `npm run test:login`.

Execute the tests for the User Profile feature using `npm run test:profile`.

After executing the tests, generate and open the Allure report using `npm run report`.

# Linting

From root folder, run `npm run wdio-lint`.

# Format cheking

From root folder, run `npm run wdio-format-check` after Git initialization.
