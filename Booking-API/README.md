# Installations

Run `npm install` from Booking-API folder.

# Test Execution Commands (from Booking-API folder)

Execute the complete test suite, with Spec and Mochawesome HTML reporters, using `npm run test-all`. The HTML report will open at the end. The HTML report is created at mochawesome-report\mochawesome.html.

Execute the complete test suite with Spec reporter, using `npm run test:spec`.

Execute the complete test suite with Mochawesome HTML reporter, using `npm run test:html`. The HTML report is created at mochawesome-report\mochawesome.html.

Execute the Ping test using `npm run test-ping`. Spec reporter will be shown, no HTML report will be generated.

Execute the Full Booking Flow (Positive Testing) using `npm run test-booking`. Spec reporter will be shown, no HTML report will be generated.

Execute the Search Flow (Positive Testing) using `npm run test-search`. Spec reporter will be shown, no HTML report will be generated.

Execute the Negative Test using `npm run test-negative`. Spec reporter will be shown, no HTML report will be generated.

# Linting

From Booking-API folder, run `npm run api-lint`.

# Format cheking

From Booking-API folder, run `npm run api-format-check` after Git initialization.
